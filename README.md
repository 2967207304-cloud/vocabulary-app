# 📚 背单词应用 (Vocabulary App)

一个基于 Vue3 + TypeScript + Pinia 开发的智能背单词应用,采用艾宾浩斯遗忘曲线算法,帮助你高效记忆单词。

## ✨ 特性

- 🎯 **智能复习** - 基于艾宾浩斯遗忘曲线的智能复习提醒
- 💾 **本地存储** - 数据保存在浏览器本地,无需后端
- 📊 **学习统计** - 可视化学习进度和成就系统
- 🎨 **简约设计** - 现代化的渐变UI,流畅的交互体验
- 📱 **响应式** - 完美支持移动端和桌面端

## 📖 使用指南

### 1. 添加单词

- 进入 **单词库** 页面
- 点击 **➕ 添加单词** 按钮
- 输入英文单词和中文释义
- 点击 **添加** 保存

### 2. 开始学习

- 进入 **今日学习** 页面
- 查看单词,点击卡片显示答案
- 根据记忆情况点击 **记住了** 或 **忘记了**
- 系统会自动调整单词的复习计划

### 3. 复习计划

- 进入 **复习计划** 页面
- 查看待复习和即将复习的单词
- 系统根据记忆曲线自动安排复习时间

### 4. 学习统计

- 进入 **学习统计** 页面
- 查看学习进度、掌握度分布
- 解锁学习成就

## 🧠 记忆曲线

应用采用艾宾浩斯遗忘曲线,复习间隔如下:

```
5分钟 → 30分钟 → 12小时 → 1天 → 2天 → 4天 → 7天 → 14天 → 30天
```

每次复习后,系统会根据你的记忆情况调整熟练度:
- **记住了**: 熟练度 +1,延长下次复习时间
- **忘记了**: 熟练度 -1,缩短下次复习时间

## 🏆 成就系统

- 🌱 **初学者** - 学习10个单词
- 🌿 **勤奋学习** - 学习50个单词
- 🌳 **词汇达人** - 学习100个单词
- ⭐ **精通大师** - 掌握20个单词
- 💪 **坚持不懈** - 复习100次
- 🔥 **连续打卡** - 连续学习7天

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **状态管理**: Pinia
- **路由**: Vue Router
- **构建工具**: Vite
- **数据持久化**: LocalStorage (通过 pinia-plugin-persistedstate)

## 💡 核心功能说明

### 单词管理 (vocabulary.ts)

- `addWord()` - 添加新单词
- `deleteWord()` - 删除单词
- `updateWord()` - 更新单词信息
- `recordStudy()` - 记录学习情况

### 智能算法

- `calculateNextReview()` - 计算下次复习时间
- `proficiency` - 熟练度等级 (0-5)
- `REVIEW_INTERVALS` - 复习间隔配置
