━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```
┌────────────────────────────────────┐
│ 第1天 - 今天                  [✕] │
├────────────────────────────────────┤
│ 共 25 个单词 | 已学习 0 个        │

- 关闭弹窗
- 点击"开始学习"按钮
- 或直接进入"今日学习"页面

#### 第4步：完成第1天后
日历更新为：

```
第1天      第2天      第3天
今天       明天       11/7
25/25     0/25       0/25
[━━━━━]  [░░░░░]    [░░░░░]
✅已完成  📖今天     ⏳待学习

#### 第5步：查看任意一天的单词
- 想看明天要学什么？点击"第2天"
- 想复习昨天的单词？点击"第1天"
- 想提前预习？点击任意未来的日期
