# 🎉 新功能说明

## 📊 学习计划功能

### ✨ 主要改进

#### 1. 每日学习限制调整
- 从每天20个新单词调整为 **每天25个新单词**
- 更合理的学习节奏，适合大量单词的学习

#### 2. 学习计划页面（原"复习计划"）

新的学习计划页面提供完整的学习规划视图：

**📚 整体学习计划**
- 总单词数统计
- 未学习单词数量
- 每天学习量（25个）
- **预计完成天数** - 自动计算完成所有单词需要多少天
- 整体进度条 - 可视化显示已完成的百分比

**🎯 今日学习目标**
- 显示当前是第几天学习
- 今日计划学习数量
- 已完成数量
- 剩余数量
- 今日完成度百分比
- 快速进入学习按钮

**🔄 复习计划**
- 待复习单词数量
- 24小时内即将复习的单词
- 基于艾宾浩斯遗忘曲线的智能复习

#### 3. 首页学习目标显示

首页新增功能：
- 显示"第 X 天"学习进度
- 今日目标进度条（已学/计划）
- 学习计划提示："还有 X 个单词未学习，按计划需要 X 天完成"
- 实时更新的今日学习统计

### 📈 使用示例

假设你导入了 **500个单词**：

1. **学习计划页面会显示**：
   - 总单词数：500
   - 未学习：500
   - 每天学习：25个
   - **预计天数：20天** ✨

2. **每天学习进度**：
   - 第1天：学习25个新单词（0→25）
   - 第2天：学习25个新单词（25→50）+ 复习昨天的
   - 第3天：学习25个新单词（50→75）+ 复习前两天的
   - ...
   - 第20天：学习最后25个（475→500）+ 复习之前的

3. **首页会显示**：
   ```
   📖 今日学习 - 第 5 天
   
   🎯 今日目标
   已完成 15/25
   [进度条 60%]
   计划学习: 25 个新单词
   剩余: 10 个
   
   💡 还有 375 个单词未学习，按计划需要 15 天完成
   ```

### 🎯 学习策略建议

1. **查看学习计划**
   - 每天先进入"学习计划"页面
   - 了解整体进度和今日目标
   - 保持学习动力

2. **按计划学习**
   - 每天完成25个新单词的目标
   - 不要贪多，遵循计划
   - 及时复习待复习的单词

3. **跟踪进度**
   - 关注整体进度百分比
   - 查看预计完成天数
   - 为自己的进步感到骄傲！
