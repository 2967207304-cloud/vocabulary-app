## 📱 使用流程

### 第一次使用
1. 进入"批量导入"页面
2. 粘贴500个单词（示例）
3. 点击导入

### 每天学习
1. 打开应用，看到：
   ```
   📖 今日学习 - 第 1 天
   🎯 今日目标: 0/25
   💡 还有 500 个单词未学习，按计划需要 20 天完成
   ```

2. 进入"学习计划"页面，看到：
   ```
   📚 整体学习计划
   总单词数: 500
   未学习: 500
   每天学习: 25 个
   预计天数: 20 天 ⭐⭐⭐
   
   🎯 今日学习目标 - 第 1 天
   计划学习: 25 | 已完成: 0 | 剩余: 25
   [进度条: 0%]
   ```

3. 开始学习，完成25个后：
   ```
   🎉 太棒了！今天的学习目标已完成！
   ```

### 持续学习
- 第2天：第2天/20天，已完成25个
- 第3天：第3天/20天，已完成50个
- ...
- 第20天：第20天/20天，已完成500个 🎊
