# 🔄 页面跳转功能说明

## ✨ 更新内容

学习日历现在改为**页面跳转**方式，不再使用弹窗！

### 🎯 使用方式

#### 在学习计划页面
1. 进入"学习计划"页面
2. 看到学习日历，每个日期都是可点击的卡片
3. **点击任意日期卡片**
4. **跳转到该天的单词列表页面** ✨

### 📱 新的单词列表页面

点击日期后，会打开一个全新的页面，包含：

#### 1. 页面头部
```
← 返回    第X天 - 日期
```
- 返回按钮：点击返回学习计划页面

#### 2. 统计信息卡片
```
┌──────────┬──────────┬──────────┬──────────┐
│ 📚 25   │ ✅ 15   │ ⏳ 10   │ 📊 60%  │
│ 总单词数 │ 已学习   │ 未学习   │ 完成度   │
└──────────┴──────────┴──────────┴──────────┘
```

#### 3. 进度条
```
[━━━━━━░░░░░░░░] 60%
```

#### 4. 单词列表
每个单词卡片显示：
```
┌────────────────────────────────────────┐
│ ① natural                      🔊  ✅ │
│    [ˈnætʃrəl] adj.                    │
│    自然的，天然的                      │
│    ┌──────────────┐                  │
│    │ 一般 复习3次 │                  │
│    └──────────────┘                  │
└────────────────────────────────────────┘
```

包含：
- 序号（彩色圆圈）
- 英文单词
- 发音按钮 🔊
- 音标
- 词性
- 中文释义
- 学习状态（✅已学习 / ⏳未学习）
- 熟练度和复习次数（已学习的单词显示）

#### 5. 操作按钮
根据不同状态显示不同按钮：

**今天的单词：**
```
┌──────────────────────────┐
│  开始学习这些单词 →      │
└──────────────────────────┘
```

**已完成的单词：**
```
┌──────────────────────────┐
│  复习这些单词 →          │
└──────────────────────────┘
```

**未来的单词：**
```
┌──────────────────────────┐
│  ⏳ 等待到这一天         │
└──────────────────────────┘
```

### 📖 使用示例

#### 场景1：查看今天要学的单词

1. 进入"学习计划"页面
2. 点击标记为"📖 今天"的卡片（蓝色背景）
3. 跳转到单词列表页面
4. 看到25个单词，可以：
   - 点击🔊听每个单词的发音
   - 查看音标、词性、释义
   - 看到已经学了多少个
5. 点击"开始学习这些单词 →"按钮
6. 进入学习页面开始学习

#### 场景2：预习明天的单词

1. 在学习计划页面
2. 点击"第2天"的卡片（橙色边框）
3. 跳转后看到明天要学的25个单词
4. 全部标记为"⏳ 未学习"
5. 可以提前：
   - 听发音，熟悉读音
   - 看释义，提前了解
6. 点击"← 返回"回到学习计划

#### 场景3：复习之前学过的单词

1. 在学习计划页面
2. 点击标记为"✅ 已完成"的卡片（绿色背景）
3. 跳转后看到那天学过的25个单词
4. 全部标记为"✅ 已学习"
5. 可以：
   - 快速复习所有单词
   - 看到自己的熟练度
   - 听发音巩固记忆
6. 点击"复习这些单词 →"进入学习模式

### 🆚 对比：弹窗 vs 页面跳转

| 特性 | 弹窗方式 | 页面跳转（新） |
|------|---------|---------------|
| 空间 | 受限 | 整个页面，更宽敞 ✅ |
| 导航 | 需要关闭弹窗 | 浏览器后退/返回按钮 ✅ |
| 体验 | 可能遮挡内容 | 独立页面，信息完整 ✅ |
| 分享 | 无法分享链接 | 可以直接分享URL ✅ |
| 书签 | 不支持 | 可以收藏特定天的单词 ✅ |
| 滚动 | 弹窗内滚动 | 整页滚动更自然 ✅ |

### 🔗 URL 格式

访问特定天的单词列表：
```
/day/1   - 第1天的单词
/day/2   - 第2天的单词
/day/10  - 第10天的单词
```

可以直接在浏览器输入URL访问！

### 💡 使用技巧

1. **浏览器后退**
   - 不需要点击"返回"按钮
   - 直接按浏览器的后退键也可以

2. **收藏重要的天**
   - 如果某一天的单词特别重要
   - 可以在浏览器收藏该页面
   - 例如：收藏 `/day/5` 

3. **在新标签页打开**
   - 按住 Ctrl（Windows）或 Cmd（Mac）
   - 点击日期卡片
   - 在新标签页中打开，方便对比

4. **直接分享**
   - 想和朋友分享某天的单词？
   - 直接发送页面链接即可
   - 例如：`http://localhost:5173/day/3`

### 🎉 优势

1. **更大的空间** - 不受弹窗尺寸限制
2. **更好的体验** - 独立页面更专注
3. **可以收藏** - 重要的单词列表可以加书签
4. **可以分享** - URL可以直接分享给他人
5. **更自然的导航** - 符合网页浏览习惯
