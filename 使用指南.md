# 📚 背单词应用 - 使用指南

## 🎯 功能特点

### 1. 批量导入单词
- 支持从文本批量导入单词
- 每天自动限制学习25个新单词
- 支持音标和词性信息

### 2. 智能复习系统
- 基于艾宾浩斯遗忘曲线的复习计划
- 根据记忆情况自动调整复习时间
- 6个熟练度等级：新词、生疏、一般、熟悉、掌握、精通

### 3. 单词发音功能
- 点击🔊按钮即可听单词发音
- 使用浏览器内置语音合成API
- 支持英式和美式发音

## 📖 快速开始

### 批量导入单词

1. 点击导航栏的"批量导入"
2. 按照指定格式粘贴单词列表：
   ```
   单词 [音标] 词性. 中文释义
   ```
   
   示例：
   ```
   natural [ˈnætʃrəl] adj. 自然的，天然的
   scenery [ˈsiːnəri] n. 景色，风景
   ```

3. 点击"预览"按钮查看解析结果
4. 确认无误后点击"导入单词"

### 查看学习计划

1. 进入"学习计划"页面，可以看到：
   - **整体学习计划**：总单词数、未学习数量、每天学习量、预计完成天数
   - **今日学习目标**：当前第几天、计划学习数量、已完成数量、剩余数量
   - **学习进度条**：可视化显示整体进度和今日进度
   - **复习计划**：待复习和即将复习的单词

### 每日学习

1. 进入"今日学习"页面
2. 页面顶部会显示：
   - 当前是第几天学习
   - 今日目标进度（已学/计划）
   - 总学习计划提示（还需多少天完成）
3. 系统会自动提供：
   - 需要复习的单词（基于遗忘曲线）
   - 今日新学单词（每天最多25个）
4. 点击卡片查看释义
5. 点击🔊按钮听发音
6. 根据记忆情况选择"记住了"或"忘记了"

### 单词管理

在"单词库"页面可以：
- 查看所有单词
- 查看单词的熟练度和复习信息
- 点击🔊听单词发音
- 手动添加单词
- 删除单词

## 📝 导入格式说明

支持以下格式（按灵活度排序）：

1. **完整格式**（推荐）：
   ```
   natural [ˈnætʃrəl] adj. 自然的，天然的
   ```

2. **无词性**：
   ```
   natural [ˈnætʃrəl] 自然的，天然的
   ```

3. **无音标**：
   ```
   natural adj. 自然的，天然的
   ```

4. **最简格式**：
   ```
   natural 自然的，天然的
   ```

## 💡 使用建议

1. **每天坚持学习**
   - 系统会自动限制每天25个新单词，避免过度学习
   - 及时复习待复习的单词
   - 在"学习计划"页面查看整体进度，保持动力

2. **充分利用发音功能**
   - 学习时多听几遍发音
   - 可以跟读练习

3. **诚实评价记忆情况**
   - "记住了"和"忘记了"会影响下次复习时间
   - 如实反馈才能获得最佳学习效果

4. **定期查看统计**
   - 在"学习统计"页面了解学习进度
   - 掌握单词数量和学习趋势

## 🔧 浏览器兼容性

- 推荐使用 Chrome、Edge、Safari 等现代浏览器
- 语音功能需要浏览器支持 Web Speech API
- 数据会自动保存在浏览器本地存储中
