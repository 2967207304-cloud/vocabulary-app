# 🚀 快速启动指南

## 步骤1: 确认环境

确保已安装 Node.js (建议 v16 或更高版本)

```bash
node --version
npm --version
```

## 步骤2: 安装依赖(如果还没安装)

在项目目录下运行:

```bash
npm install
```

## 步骤3: 启动开发服务器

```bash
npm run dev
```

## 步骤4: 打开浏览器

看到类似以下输出:

```
  VITE v5.2.0  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

在浏览器中访问: **http://localhost:5173/**

## 🎉 开始使用

1. 进入 **单词库** 页面
2. 点击 **➕ 添加单词**
3. 添加你的第一个单词
4. 返回 **今日学习** 开始背单词!
