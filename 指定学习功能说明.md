# 🎯 指定学习特定天单词功能

## ✨ 功能说明

现在点击"开始学习这些单词"按钮，会**只学习当前这一天的25个单词**，而不是学习所有待学习的单词！

## 🔄 工作流程

### 1. 查看某天的单词列表

```
学习计划页面 → 点击"第3天"卡片 → 跳转到单词列表页面
```

显示：
- 第3天的25个单词
- 统计信息
- 进度条
- 操作按钮

### 2. 点击"开始学习这些单词"

```
单词列表页面 → 点击"开始学习这些单词 →" 
→ 跳转到学习页面
→ 只学习第3天的这25个单词 ✅
```

### 3. 学习页面显示

**页面标题会变化：**
- 普通模式：`📖 今日学习 - 第 X 天`
- 指定模式：`📖 学习第 3 天的单词` ✨

**学习进度：**
```
正在学习：1 / 25
（只有这一天的25个单词）
```

## 📊 对比

### 之前的行为 ❌
```
点击"第3天" → 查看单词 → 点击"开始学习"
→ 学习所有待学习的单词（可能有100+个）
→ 包括第3天的 + 其他天的
```

### 现在的行为 ✅
```
点击"第3天" → 查看单词 → 点击"开始学习这些单词"
→ 只学习第3天的25个单词
→ 学完这25个就完成了
```

## 🎯 使用场景

### 场景1：专注学习某一天

**需求：** 今天想专门学习第5天的单词

1. 进入"学习计划"
2. 点击"第5天"卡片
3. 查看这25个单词
4. 点击"开始学习这些单词 →"
5. 专注学习这25个，不会被其他单词打扰 ✅

### 场景2：补学之前的单词

**需求：** 第2天有10个单词没学完，想补学

1. 进入"学习计划"
2. 点击"第2天"卡片（显示15/25已完成）
3. 看到哪些单词还没学
4. 点击"开始学习这些单词 →"
5. 只学习第2天的这25个单词
6. 重点关注那10个未学的 ✅

### 场景3：复习特定一天

**需求：** 觉得第1天的单词有点忘了，想复习

1. 进入"学习计划"
2. 点击"第1天"卡片（✅已完成）
3. 查看这25个单词
4. 点击"复习这些单词 →"
5. 再次学习第1天的25个单词 ✅

## 💡 技术实现

### URL 参数传递

点击按钮后，URL变为：
```
/?day=3
```

含义：学习第3天的单词

### 学习页面逻辑

```javascript
如果 URL 中有 day 参数：
  → 只加载第 X 天的25个单词
否则：
  → 加载默认学习列表（待复习 + 今日新词）
```

### 页面标题

```javascript
如果有 day 参数：
  显示："📖 学习第 X 天的单词"
否则：
  显示："📖 今日学习 - 第 X 天"
```

## 🔍 详细示例

### 示例1：学习第3天

1. **在学习计划页面**
   - 看到"第3天"卡片
   - 显示：0/25 ⏳待学习

2. **点击卡片，跳转到**
   ```
   /day/3
   ```
   - 显示第3天的25个单词列表

3. **点击"开始学习这些单词 →"，跳转到**
   ```
   /?day=3
   ```
   - 页面标题：`📖 学习第 3 天的单词`
   - 进度显示：`1 / 25`

4. **学习过程**
   - 第1个单词：natural
   - 第2个单词：scenery
   - ...
   - 第25个单词：contaminate
   - 学完！✅

5. **回到学习计划**
   - "第3天"卡片变成：25/25 ✅已完成

### 示例2：默认学习模式

1. **直接进入学习页面**
   ```
   /
   ```
   - 页面标题：`📖 今日学习 - 第 5 天`
   - 学习列表：待复习的 + 第5天的新词（可能>25个）

2. **完成学习**
   - 先复习之前学过的单词
   - 再学习今天的25个新词

## ✅ 优势

1. **专注学习** 
   - 一次只学25个
   - 不会被大量单词淹没

2. **灵活安排**
   - 可以跳着学
   - 可以重点攻克某一天

3. **明确进度**
   - 知道这25个单词的学习进度
   - 学完就是学完，成就感强

4. **补学方便**
   - 哪天没学完，可以单独补
   - 不影响其他天的学习
